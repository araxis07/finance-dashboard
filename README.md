# 💰 Personal Finance Dashboard

A modern frontend web application for tracking income, expenses, and financial insights.
Designed with scalability in mind, this project simulates real-world fintech features including smart file uploads and AI-ready data parsing.

---

## 🚀 Features

* 📊 Financial dashboard (balance, income, expense)
* 🧾 Transaction management (add / view)
* 📈 Expense visualization (charts by category)
* 📂 File upload UI (PDF, Excel, Images)
* 🤖 Mock AI parsing system (ready for OCR + AI integration)
* 💾 Local state persistence (Zustand + localStorage)

---

## 🛠️ Tech Stack

* Next.js (App Router)
* Tailwind CSS
* Zustand (state management)
* Recharts (data visualization)

---

## 📸 Screens

* Dashboard overview
* Transaction list
* Upload interface

---

## 🧠 Future Improvements
